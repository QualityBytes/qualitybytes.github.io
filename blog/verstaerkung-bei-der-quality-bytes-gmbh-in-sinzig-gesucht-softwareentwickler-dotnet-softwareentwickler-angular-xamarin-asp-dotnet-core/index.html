Verstärkung als Softwareentwickler .NET, Frontend-Developer Angular, Xamarin Developer oder ASP.NET Core Entwickler gesucht

Verstärkung als Softwareentwickler .NET, Frontend-Developer Angular, Xamarin Developer oder ASP.NET Core Entwickler gesucht

  • André Krämer
  • 19.03.2019
  • .NET Angular ASP .NET Core Xamarin
  • Lesezeit: 1 Minute/n

Das könnte dein neuer Schreibtisch sein

Für unser Team in Sinzig am Rhein, zwischen Bonn und Koblenz, suchen wir ab sofort mehrere Softwareentwickler. Wir entwicklen in einem Team von vier Entwicklern spannende Kundenlösungen im Web- und Mobile-Umfeld.

Wir setzen dabei auf moderne Technologien, Werkzeuge und Vorgehensweisen, wie z. B.

  • ASP.NET Core
  • Angular
  • Xamarin
  • Azure DevOps
  • git
  • Unit Testing
  • Typescript
  • C#

Aktuell haben wir mehrere Stellen zu besetzen. Unter anderem suchen wir drei .NET Entwickler, die wahlweise ASP.NET Core oder Xamarin oder beides entwickeln sowie einen Frontend Entwickler mit einem Schwerpuntk auf Angular.

Wenn du also Lust hast an innovativen Projekten in einem familiären Team mit flachen Hierarchien, geregelten Arbeitszeiten und guten Weiterentwicklungsmöglichkeiten zu arbeiten, dann guck dir doch mal unsere offenen Stellen an und bewirb dich.

Über den Autor

André Krämer

Dipl.-Inform. (FH) André Krämer, Msc. ist Geschäftsführer der Quality Bytes GmbH. Der mehrfache Microsoft MVP ist seit 1997 in der Softwareentwicklung tätig und hat sich 2002 auf die Entwicklung mit Microsoft .NET spezialisiert.

Seine Schwerpunkte sind Xamarin, ASP.NET, Angular, Azure DevOps, TX Text Control und die Analyse von Memory Leaks und Performance Problemen.

Er ist regelmäßiger Autor für linkedIn Learning, wo er bereits mehr als 25 Videotrainings veröffentlicht hat.
